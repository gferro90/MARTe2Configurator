Detailed Description

The GAM scheduler interface.

Provides methods to link the scheduler to the threads to be executed and the mechanisms to allow a swiftly change of state.

The syntax in the configuration stream has to be:

+Scheduler = {
Class = Scheduler_name ...
TimingDataSource = "Name of the TimingDataSource" }
and it has to be contained in the [RealTimeApplication] declaration.
