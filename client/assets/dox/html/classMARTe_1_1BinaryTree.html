Detailed Description

template<typename T, typename HashObject = DjbHashFunction>
class MARTe::BinaryTree< T, HashObject >

Implementation of a simple binary tree.

For each element to be added to the list, an integer key is computed using the D. J. Bernstein hash function (see HashFunction). The element and the computed key are inserted keeping the list sorted by keys. This allows to speed up the search operation to log(N) using the binary search.
