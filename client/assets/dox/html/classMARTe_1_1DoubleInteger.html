Detailed Description

template<typename T>
class MARTe::DoubleInteger< T >

Template class which represents an integer which duplicates the size of the also integer template T type.

Template Parameters
Tthe integer type of the upper and lower part of the double integer defined by this class. This integer type must provide at list the same operators that the class provides.
Remarks
The interpretation of the endianity is system dependent.
