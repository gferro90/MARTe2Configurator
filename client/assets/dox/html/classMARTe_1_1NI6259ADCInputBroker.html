Detailed Description

An input broker for NI6259 which copies from the latest buffer available.

The NI6259 DataSourceI copies the data using DMA to one of two buffers. This NI6259ADCInputBroker copies the data back to the GAM memory from the latest complete buffer written by the NI6259ADC.
