bool MARTe::NiSyncClockGenerator::Initialise ( StructuredDataI &  data)
virtual

Initialises the component.

The user must define the following parameters: Terminal = the terminal name ClockPeriod = the clock period in nano seconds DutyCycle = the duration of the duty cycle in nano seconds Phase = the phase with respect the board time seconds. @detail The user can optionally define the following parameters: StartTimeNanos = how much time to wait before starting the clock (default: start immediately) StopTimeNanos = when to stop the clock (default: infinite)
