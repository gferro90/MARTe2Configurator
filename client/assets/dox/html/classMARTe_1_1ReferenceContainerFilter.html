Detailed Description

Defines the filtering criteria for the ReferenceContainer searches.

This class needs to be specialised to implement different search criteria. The find function of the ReferenceContainer will walk the list and query, by calling Test, if a given node has been found.

The different types of searching modes are also stored by this class.

Every time the Test function is to return it should call IncrementFound so that the number of occurrences found is correctly tracked.
