Detailed Description

Input broker for the RealTimeThreadSynchronisation DataSourceI.

A MemoryMapInputBroker which will store in memory the required number of samples copies of the DataSourceI memory. It will lock in Execute until the required number of samples are added by calling the AddSample method.
