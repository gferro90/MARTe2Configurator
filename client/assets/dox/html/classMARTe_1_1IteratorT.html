Detailed Description

template<typename T>
class MARTe::IteratorT< T >

Template version of the Iterator interface.

This interface is the template version of the interface functor Iterator, parameterising the type of the object on which the Do method will apply an specific action.

Template Parameters
Tthe type of the object on which an action will be applied by the Do method.
