Detailed Description

Namespace which defines constants and mathematical functions using fast processor instructions.

The functions of this namespace are those mathematical operations that can be optimized on target architecture.

The target architecture is the one of the machine where the code is being executed. Most of the implementation is therefore delegated to a version which will be specific for each type of supported architecture.

template<typename Type >
Type MARTe::FastMath::SquareRoot ( const Type  x)
inline

Generic template implementation to compute square root of number.

A template specialisation is provided for integer number.

Parameters
[in]xis argument to compute the square root of.
Returns
the truncated (in the same type) square root of x. If applied to a negative signed integer, the function typecasts its additive inverse to unsigned and computes the square root of the casted integer. If applied to a negative float, the function computes the square root of the additive inverse.
