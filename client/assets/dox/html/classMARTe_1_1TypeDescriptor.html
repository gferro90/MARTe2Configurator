Detailed Description

A structure Used to describe the type pointed to by a pointer.

Depending on the first bit isStructuredData it may contain a code identifying a structure or the remaining bits can be used to identify a specific basic type.

Basic types are integers 8-64 bit, floats, doubles, char pointers and void pointers.

Remarks
The TypeDescriptor is internally represented as a 16-bit bitfield-like union with one of the following structures (depending on the type described, if a basic one, or a structured one):
isStructuredData isConstant type numberOfBits
1 1 4 10
isStructuredData isConstant structuredDataIdCode
1 1 14
