Detailed Description

Defines the Function elements (aka GAMs) to be executed in a real-time thread.

The syntax in the configuration stream shall be: +RealTimeThread_name = {
Class = RealTimeThread
Functions = { GAM1_name, GAMGroup2_name, ... } CPUs = 0xf //CPU affinity mask for the thread. Optional parameter. StackSize = 32768 //Stack size for the thread. Optional parameter. }

◆ Initialise()

bool MARTe::RealTimeThread::Initialise ( StructuredDataI &  data)
virtual

Reads the array with the GAM names to be executed by this thread.

The following fields must be defined:

Functions = { function1_path, function2_path, ... }

Each element must be the path of the function to be executed by this thread with respect to the position of the "Functions" node.

The following fields can be defined

StackSize = (the memory stack size in byte to be associated to the this thread) CPUs = cpu mask where this thread is preferable to be executed (i.e 0x1 means the first cpu, 0x2 means the second, 0x3 first and second, ...).

The default value for StackSize is THREADS_DEFAULT_STACKSIZE, while for CPUs is ProcessorType::GetDefaultCPUs().

Parameters
[in]datais the StructuredData to be read from.
Returns
true if the parameters Functions is declared in data and the number of elements in Functions is greater than zero.
Postcondition
GetFunctions() != NULL && GetNumberOfFunctions() > 0
