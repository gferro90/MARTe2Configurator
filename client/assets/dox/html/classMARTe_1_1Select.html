Detailed Description

Class for monitoring simultaneously I/O events from devices like files, sockets, console, and so on.

The Select class provides a set of functions to monitor several I/O (files, sockets, console...) events at the same time. The class allows to add, remove or clear handles in read, write or exception mode. Once the handles are selected, the WaitUntil() function starts to monitor them returning when an even occurs. The IsSet() function can be used to know which events were triggered.
