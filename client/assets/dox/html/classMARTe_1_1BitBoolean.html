Detailed Description

template<typename baseType, uint8 bitOffset>
class MARTe::BitBoolean< baseType, bitOffset >

Boolean shifted type.

This type could be used in an union obtaining the same effect of a one bit boolean in a structure.

Template Parameters
baseTypethe standard type which is used as a base for the new type
bitOffsetthe actual bit offset of the new type
Remarks
bitOffset must be less than the number of bits of baseType.
