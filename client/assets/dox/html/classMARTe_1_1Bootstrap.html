Detailed Description

Bootstraps the main arguments and provides the (environment specific) execution context (see Run).

The implementation of the class is operating system specific.

ErrorManagement::ErrorType MARTe::Bootstrap::ReadParameters ( int32  argc,
char8 **  argv,
StructuredDataI &  loaderParameters 
)

Translate the main (argc and argv) parameters into a StructuredDataI.

Parameters
[in]argcthe number of arguments provided in the main.
[in]argvthe list of arguments provided in the main.
[out]loaderParametersthe list of parsed parameters:
  • Loader: the type of loader class to be used;
  • Filename: the name of the file to be load;
  • DefaultCPUs: sets the threads defaults CPUs (see ProcessorType::SetDefaultCPUs);
  • Parser: the type of parser to be parse the configuration as one of:cdb, xml and json;
  • MessageDestination (optional): the name of the Object that will receive the message upon program Start (see Loader::Initialise);
  • MessageFunction (optional): the name of the Function to be called in the MessageDestination (see Loader::Initialise);
  • FirstState (optional): the name of the FirstState in a RealTimeApplication (see RealTimeLoader::Initialise).
Returns
ErrorManagement::NoError if all the compulsory parameters can be read from the argv and written into the loaderParameters. A specific ErrorType otherwise.
