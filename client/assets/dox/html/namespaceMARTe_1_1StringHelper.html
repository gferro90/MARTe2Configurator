Detailed Description

Namespace which defines functions for manipulating C strings.

These functions allows to query and manipulate C strings, i.e. null-terminated arrays of characters (char8 *).

These kind of functions are particularly useful in the implementation of more abstract string classes, and also for stream classes which usually need to manage buffers of chars.
