Detailed Description

Interface which defines a functor responsible of doing an action on a LinkedListable instance.

The classes that realise this interface will implement a specific action assuming that the parameter will be an object of type LinkedListable.

Remarks
In order to apply an action on all items of a list, the user will have to call the Do method for each element of the list.
