Detailed Description

An helper class to serialise the contents of an AnyType (i.e.

of the data pointed by an AnyType.GetDataPointer()). This class allows to associate a name and a reference to an AnyType. It holds and manages a memory space to store a copy of the data of the serialised AnyType.

bool MARTe::AnyObject::Serialise ( const AnyType &  typeIn)

Serialises an AnyType.

The input AnyType typeIn will be copied and its contents (provided by typeIn.GetDataPointer()) copied into a memory space managed by this AnyObject. This memory will be freed by the destructor of the AnyType.

Parameters
[in]typeInthe AnyType to be serialised.
Returns
true if all the memory allocation and copy operations are successful.
