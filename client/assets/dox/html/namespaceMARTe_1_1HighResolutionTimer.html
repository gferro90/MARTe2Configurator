Detailed Description

Namespace which defines functions for high resolution timing using the cpu clock.

The functions of this namespace offer 2 kind of services:

  • Functions for querying the current time, like a counter or a timestamp.
  • Functions for getting timing related properties of the processor, like frequency and period.

These functions are particularly useful in the implementation of functionality that require a high resolution timer precision, like in sempahores.

The implementation of these functions can relay on services provided by operating system or by the processor itself.
