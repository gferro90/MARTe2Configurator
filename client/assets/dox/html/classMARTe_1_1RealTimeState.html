Detailed Description

A container of one (and only one) ReferenceContainer of RealTimeThread elements.

The RealTimeState lists the threads which are allowed in any given state of the application. Switching state will enable different threads to be executed.

The syntax in the configuration stream shall be: 

State_name = {
    Class = RealTimeState
    +Threads = {
        Class = ReferenceContainer
        RealTimeThread_name = {
            Class = RealTimeThread
            ...
        }
        ...
     }
}
