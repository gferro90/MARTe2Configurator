Detailed Description

Generic class to implement any kind of waveform.

This class contains the generic common functions and variables to implement any waveform signal.

Three different GAMs are foreseen to use this generic Class: WaveformSinGAM WaveformPointsDefGAM WaveformChirpGAM

This class implements the common functionalities that are common to all the derived classes, which are basically the setup of the trigger mechanism and the selection of the output type.

The trigger time must be specified in seconds and the type shall be float64.

bool MARTe::Waveform::Initialise ( StructuredDataI &  data)
virtual

Initialise the common variables from a configuration file.

Initialise and verifies the trigger mechanism. This function checks the size of the startTriggerTime and stopTriggerTime and checks that startTriggerTime[i] < stopTriggerTime[i]. An invalid trigger time sequence does not give an error only a warning is launched, however the trigger mechanism is disabled and the output always will be on with the expected defined waveform.

Parameters
[in]datais the GAM configuration file
Returns
true if MARTe::GAM::Initialise(StructuredDataI &data) exits without errors.

Reimplemented in MARTe::WaveformChirp, MARTe::WaveformSin, and MARTe::WaveformPointsDef.

bool MARTe::Waveform::Setup ( )
virtual

Setup the input/output variables.

Initialises the input and output pointers and verifies the number of inputs and output sizes. In particular it verifies that a TimeSignal was set with one and only one element and that the type is uint32 or int32.

Returns
