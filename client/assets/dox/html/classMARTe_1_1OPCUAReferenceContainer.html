Detailed Description

Extension of the base ReferenceContainer class in order to add functionalities for OPCUA Server configuration.

Parent Class of OPCUANode and OPCUAObject classes. It extends the base class ReferenceContainer adding set and get common function to OPCUA Nodes and Objects.
