Detailed Description

MessageI queued implementation.

Messages consumed by this MessageI are processed in the context of a thread. A QueueingMessageFilter is installed (MessageI::InstallMessageFilter) and the thread blocks until a new message is consumed by this queue (QueueingMessageFilter::GetMessage). This message is then propagated to all the filter that were added to this QueuedMessageI (see InstallMessageFilterInQueue).
