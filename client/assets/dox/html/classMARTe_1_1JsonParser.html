Detailed Description

Concrete class for MARTe::ParserI abstract class, configured for streams of characters encoded in JSON.

This class is a concrete class for MARTe::ParserI providing the actual lexical elements and parsing rules for interpreting a stream of characters encoded in JSON.

Each instance of the parser is bound when it is constructed, with all the objects involved in the parsing analysis, as follows:

  • An input stream of characters that contains the serialization of a hierarchy of objects, encoded into JSON.
  • An output structured data store where the parser will build the in- memory objects defined into the input stream of characters.
  • An output stream of characters where the parser will write all the errors found on the input stream of characters.

All the instances of the parser use the lexical elements defined in MARTe::JsonGrammar and apply the parsing rules of the following grammar:

1 EXPRESSION:
2  STRING --> GetNodeName() \: VARIABLE __AddLeaf
3  STRING __CreateNode \: \\{ { EXPRESSION }+ \\} __EndBlock
4 
5 VARIABLE:
6  SCALAR
7  VECTOR
8  MATRIX
9 
10 SCALAR:
11  TOKEN __AddScalar
12 
13 VECTOR
14  \\[ SCALAR ... \\] __EndVector
15 
16 MATRIX:
17  \\[ VECTOR ... \\] __EndMatrix
18 
19 TOKEN:
20  STRING
21  NUMBER

Note: This grammar is written in the SLK language and refers to functions declared in MARTe::ParserI.
