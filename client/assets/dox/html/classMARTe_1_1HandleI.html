Detailed Description

Interface which offers access to read and write filesystem handles.

This interface offers methods for getting system dependent read and write handles for device classes like files, sockets and consoles.

Remarks
When the class that implements this interface has only one handle, then is is expected that both methods (read and write handles) will return the same handle.
