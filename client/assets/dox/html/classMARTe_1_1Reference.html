Detailed Description

Shared pointer implementation.

The Reference class is a smart pointer implementation where the shared ownership of an underlying object is enabled. This guarantees that the life-cycle of this object is safely managed and in particular that the object is destructed when it is no longer referenced.

The Reference might also own no object, in which case the function IsValid will return false.
