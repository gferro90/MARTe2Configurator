Detailed Description

Input MemoryMapBroker implementation.

This class copies all the signals declared on a MemoryMapBroker from the DataSourceI memory to the GAM memory.

bool MARTe::MemoryMapInputBroker::Execute ( )
virtual

Sequentially copies all the signals from the DataSourceI memory to the GAM memory.

This implementation supports multi-state buffers and will query the DataSource for the GetCurrentStateBuffer.

Returns
true if all copies are successfully performed.

Implements MARTe::ExecutableI.

Reimplemented in MARTe::MemoryMapSynchronisedInputBroker.
