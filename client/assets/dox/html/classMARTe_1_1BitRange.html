Detailed Description

template<typename baseType, uint8 numberOfBits, uint8 bitOffset>
class MARTe::BitRange< baseType, numberOfBits, bitOffset >

A number with configurable size and bit offset.

Using these types into an union allows to have the same effect of a struct with bit fielded attributes.

Template Parameters
baseTypethe standard type which is used as a base for the new type
numberOfBitsthe actual bit size of the new type
bitOffsetthe actual bit offset of the new type
Remarks
numberOfBits + bitOffset must be minor than the bit size of baseType.

void MARTe::BitRange< baseType, numberOfBits, bitOffset >::operator= ( inputType  input)

Copy operator.

Since this type could be used in an union, this function change only the range of bits specified leaving untouched the bits outside the range.

The input argument could be another type. If its value is greater than the maximum possible value or lower than the minimum possible value, the value assigned will be saturated.
