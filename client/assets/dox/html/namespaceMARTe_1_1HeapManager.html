Detailed Description

Namespace which defines functions for the heap memory management.

The collection of functions of this namespace act as a stateful singleton object, because they maintain an implicit database of heaps allowing to add, remove, and find heaps. They also act as a fa√ßade for allocating and freeing memory managed by the stored heaps.

Duplicates a memory section into a new area from the specified heap.

If size != 0 the function duplicates the specified number of bytes of the memory area pointed by data.
If size = 0 the function considers the memory area pointed by data as a zero terminated string and copies the memory until a zero is found (C string duplication).

Parameters
[in]dataThe pointer to the memory which must be copied.
[in]sizesize of memory to allocate. if size = 0 then memory is copied until a zero is found
[in]heapNamename of heap to use. default = copy on the same heap as data
Returns
The pointer to the new allocated memory which contains a copy of s.
Precondition
If size = 0 the memory area pointed by data must be a zero terminated string.

DLL_API bool MARTe::HeapManager::Free ( void *&  data)

Calls the HeapI::Free function of the HeapI object which owns the specified address.

The address in input could belong to more than one HeapI registered in the database and in this case the free of the one with the smaller range of memory will be performed.
