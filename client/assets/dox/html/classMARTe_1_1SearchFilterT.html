Detailed Description

template<typename T>
class MARTe::SearchFilterT< T >

Template version of the SearchFilter interface.

This interface is the template version of the interface functor SearchFilter, parameterising the type of the object that the Test method will check for compliance, with respect to a given searching criteria.

Template Parameters
Tthe type of the object which will be checked by the Test method.
Remarks
It is expected that T be descendant of LinkedListable.
