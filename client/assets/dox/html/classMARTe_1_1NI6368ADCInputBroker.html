Detailed Description

An input broker for NI6368 which copies from the latest buffer available.

The NI6368 DataSourceI copies the data using DMA to one of two buffers. This NI6368ADCInputBroker copies the data back to the GAM memory from the latest complete buffer written by the NI6368ADC.
