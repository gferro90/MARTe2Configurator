Detailed Description

Namespace which defines the available modes of a reference container search.

The modes are defined as constants of type uint32. They are meant to be used for configuring searches on reference containers.

Variable Documentation

const uint32 MARTe::ReferenceContainerFilterMode::PATH = 0x4u

Store all the nodes leading to a given occurrence.

if SearchAll mode (occurrence = -1) a warning will be generated and the the filter will match the first occurrence (occurrence = 1).

const uint32 MARTe::ReferenceContainerFilterMode::REMOVE = 0x8u

If set the nodes that are found are deleted.

To destroy all occurrences occurrence must be set to -1 in the constructor of ReferenceContainerSearchMode.
