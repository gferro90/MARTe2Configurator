Detailed Description

Implementation of a DataSource capable of reading and writing data from/to an external MemoryGate object whose path is defined in the configuration.

The user has to specify in the configuration the field "Link" containing the absolute path of the MemoryGate object to be linked with. Moreover the user has to specify in the configuration the flag "IsWriter" that defines if this data source has to write or reads its signals memory from/to the linked MemoryGate within the Synchronise() function.

Follows an example of configuration. 

 +InputDataSource1 = {
     Class = LinkDataSource
     Link = ExternalComponent1
     IsWriter = 0
 }
