Detailed Description

OPCUAServer class.

Manages the OPCUA Server life cycle.

A SingleThreadService which offers all the functionalities to read data from IntrospectionStructures and to create the OPCUA Address Space correctly.

All the nodes added in the AddressSpace will be of OPCUA Variable type.

The configuration syntax is (names are only given as an example): 

+OPCUAServer
    Class = OPCUA::OPCUAServer
    Port = 4840 //Optional. Default is 4840
    CPUMask = 0x4
    AddressSpace = {
        MyNodeStructure1 = {
            Type = MyIntrospectionStructure1
        }
        MyNodeStructure2 = {
            Type = MyIntrospectionStructure2
        }
        ...
        MyNode = {
            Type = float64
        }
    }
