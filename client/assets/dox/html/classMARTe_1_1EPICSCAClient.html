Detailed Description

A container of EPICSPV variables.

Provides the threading context for the EPICS CA interface.

The configuration syntax is (names are only given as an example):

+EPICS_CA = {
  Class = EPICSInterface::EPICSCAClient
  StackSize = 1048576 //Optional the EmbeddedThread stack size. Default value is THREADS_DEFAULT_STACKSIZE * 4u
  CPUs = 0xff //Optional the affinity of the EmbeddedThread (where the EPICS context is attached).
  AutoStart = 0 //Optional. Default = 1. If false the service will only be started after receiving a Start message (see Start method).
  +PV_1 = {
     Class = EPICSPV //See class documentation of EPICSPV
     ...
  }
  +PV_2 = {
     Class = EPICSPV
     ...
  }
}

ErrorManagement::ErrorType MARTe::EPICSCAClient::Execute ( ExecutionInfo &  info)

Provides the context to execute all the EPICS relevant calls.

Executes in the context of a spawned thread the following EPICS calls: ca_context_create, ca_create_channel, ca_create_subscription, ca_clear_subscription, ca_clear_event, ca_clear_channel, ca_detach_context and ca_context_destroy

Returns
ErrorManagement::NoError if all the EPICS calls return without any error.
