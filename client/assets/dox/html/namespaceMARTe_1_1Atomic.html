Detailed Description

Namespace which defines functions that are executed atomically even on multiprocessor machines.

The functions of this namespace are those needed for increment, decrement, testandset, and so on.

These methods are particularly useful in the implementation of functionality that requires protected basic operations between multiple asynchronous accesses like in a multi-thread system. These methods are also useful to implement fast threading synchronization constructors.

The target architecture is the one of the machine where the code is being executed. Most of the implementation is therefore delegated to a version which will be specific for each type of supported architecture.
