Detailed Description

Namespace which defines functions for threads management.

The collection of functions of this namespace act as a stateful singleton object, because they maintain an implicit database of thread information records, allowing to create, manage and monitor the life-cycle of any framework thread.

The default stack size associated to a thread THREADS_DEFAULT_STACKSIZE is defined in GeneralDefinitions.h for each different operating system.

enum MARTe::Threads::PriorityClassType

Available priority classes.

The thread priority is defined by a priority class and a priority level. The priority class defines the policy and the level defines the rank of the priority inside a given class.

Enumerator

Unknown priority class.

IdlePriorityClass 

Lowest priority class.

Normal priority class.

Higher than Idle.

Real-time priority class.
