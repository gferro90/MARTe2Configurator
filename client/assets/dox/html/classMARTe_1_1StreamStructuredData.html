Detailed Description

template<class Printer>
class MARTe::StreamStructuredData< Printer >

Implementation of a StructuredDataI interface wrapped over a stream.

See also
StreamStructuredDataI This class allows to use the StructuredDataI interface without need to store data in memory, but streaming out directly on a chosen stream. Only write operations are allowed. This templated version allows to choose a printer (that must be a descendant of PrinterI) to print on the stream using the database language implemented by the printer object. All the StructuredDataI read operations will return false because this object is intended to be used only to write out data. Since the write operations are directly flushed on the internal stream, it is not allowed to move or create new nodes backward in the database. For Instance in these sequence of operations:
CreateAbsolute("A.B") //allowed
CreateAbsolute("A.C") //allowed but now B is closed
CreateAbsolute("A.B.D") //not allowed: it is not possible return back into B
