Detailed Description

A basic implementation of a stream which allows reading and writing to an underlying bounded and preallocated memory location.

MARTe::StreamMemoryReference::StreamMemoryReference ( char8 *const  bufferIn,
const uint32  bufferSize 
)

Constructor from a preallocated read/write buffer.

This constructor binds the object to a memory area in read/write mode. It also assumes that the area of memory is empty and therefore the stream's size is 0.

Parameters
[in]bufferInis the char8 pointer of the preallocated read/write memory.
[in]bufferSizeis the usable size of the buffer.
Precondition
bufferIn != NULL bufferSize > 0
Postcondition
CanRead() && CanWrite() && CanSeek() && Position() == 0 && Size() == 0
Warning
The buffer pointer is an input parameter to the constructor, but the pointee will be accessed for reading and writing by the other methods of the class.
MARTe::StreamMemoryReference::StreamMemoryReference ( const char8 *const  bufferIn,
const uint32  bufferSize 
)

Constructor from a preallocated read only buffer.

This constructor binds the object to a memory area in read only mode. It also assumes that the area of memory is full and therefore the stream's size is bufferSize.

Parameters
[in]bufferInis the const char8 pointer to a preallocated read only memory.
[in]bufferSizeis the usable size of the buffer.
Precondition
bufferIn != NULL bufferSize > 0
Postcondition
CanRead() && not CanWrite() && CanSeek() && Position() == 0 && Size() == bufferSize
Warning
The buffer pointer is an input parameter to the constructor, but the pointee will be accessed (read only) by the other methods of the class.

Moves within the stream relative to current location.

Checks that the final position is >= 0 and <= UsedSize, then moves the cursor.
-If the final position is < 0 moves the cursor at the beginning.
-If the final position is > UsedSize moves the cursor at the end.
