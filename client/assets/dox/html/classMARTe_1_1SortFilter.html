Detailed Description

Interface which defines a functor responsible of answering which is the order direction of a couple of LinkedListable's instances.

The classes that realise this interface will implement the checking of the order direction into the Compare method, assuming that the parameters will be objects of type LinkedListable.

Remarks
In order to sort all the items of a list, the user will have to call the Compare for pairs of list's elements from a sorting algorithm.
