Detailed Description

This class represents the concept of a block of signals.

The signals hold by the sigblock do not have a pre-established position into the sigblock, because it depends on the size of the actual type of each one of them. So, the class offers access to the address of each signal, being responsibility of the clients to deal with the binary representation of each signal and cast it properly.
