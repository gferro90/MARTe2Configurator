Detailed Description

The GAM scheduler.

The syntax in the configuration stream has to be:

+Scheduler = {
Class = Scheduler_name ...
TimingDataSource = "Name of the TimingDataSource" +ErrorMessage = { //Optional. Fired every time there is an execution error. Name is only an example. Class = Message ... } }
