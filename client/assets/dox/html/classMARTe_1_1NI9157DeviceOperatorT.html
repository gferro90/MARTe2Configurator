Detailed Description

template<typename T>
class MARTe::NI9157DeviceOperatorT< T >

A class to allow typed read and write operations from/to a NI-9157 device.

Depending on the template type, the methods NiRead, NiWrite, NiReadFifo, NiWriteFifo, overload the relative methods of the member NI9157Device.
