Detailed Description

template<typename T>
class MARTe::ObjectBuilderT< T >

This class template is a helper for building specific versions of object builders for specific classes (the class specified as a class template parameter).

Template of ObjectBuilder implementing a default memory allocation for the new instance of a MARTe::Object descendant.

Template Parameters
Tis the typename of a class descendant of MARTe::Object, which has been registered as a class into ClassRegistryDatabase, and that provides a new operator compatible for working with HeapI as memory manager.
