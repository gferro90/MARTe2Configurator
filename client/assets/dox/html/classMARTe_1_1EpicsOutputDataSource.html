Detailed Description

A data source which puts its signals' values to an EPICS IOC.

This class is an extension of the DataSourceI class, allowing to write its signals' values to an EPICS IOC service by means of an output broker, which will call the datasource's Synchronise method every time it is executed.

Each instance of this class will create and register a public shared memory area in the OS, named "<application_name>_<datasource_name>", where the ex- change of signals will occur. It is expected that the EPICS IOC connects to the shared memory area and gets the last signals' values at their own pace.

The configuration syntax is (names are only given as an example): +ExportSignalsFromIOC = { Class = EpicsOutputDataSource }

A signal will be added for each GAM signal that writes to this instance of the DataSourceI.
