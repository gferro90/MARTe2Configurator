◆ GetMemberAttributes()

const char8 * MARTe::IntrospectionEntry::GetMemberAttributes ( ) const

Retrieves the member attributes.

The string attributes must be written in the standard MARTe configuration language and provides a series of attributes to be associated to a basic type member. Most common attributes are:

min = ... —> the minimum value
max = ... —> the maximum value
values = {...} —> the list of possible values
min_size = ... —> the minimum size
max_size = ... —> the maximum size

The functions in the class ValidateBasicType will take as input these attributes to check if the value to associate to the class member respects the constraints returned by this function.

Returns
the member attributes.

◆ GetMemberModifiers()

const char8 * MARTe::IntrospectionEntry::GetMemberModifiers ( ) const

Retrieves the member modifiers.

The string modifiers is a list of symbols:

'*' = pointer;
'C' = constant;
'[n]' = array of n elements;

The string must be a combination of '*'s and 'C's terminated by, if applicable, the specification for the array size. Assuming that the type int and that the variable is called x, some examples follow:

"**" —> double pointer ( int ** x );
"**C" —> constant double pointer ( int ** const x );
"C*C*C" —> constant pointer to constant pointer to contant integer ( int const * const * const x );
"*C*C" —> constant pointer to constant pointer ( int * const * const x );
"*C*" —> pointer to constant pointer ( int * const * x );
"[2]" —> array ( int x[2] );
"*[2]" —> array of pointers ( int *x[2] );
"[2][2]" —> double array (int x[2][2] );

Returns
the member modifiers.

◆ IsConstant()

bool MARTe::IntrospectionEntry::IsConstant ( const uint32  ptrLevel = 0u) const

Specifies if the member type is constant.

Parameters
[in]ptrLevelspecifies the pointer level to be checked in case the member is a n-th pointer.

Example for "const int ** const x":

ptrLevel == 0 --> true (the final type is constant)
ptrLevel == 1 --> false (the first pointer is not constant)
ptrLevel == 2 --> true (the second pointer is constant)
If ptrLevel > GetMemberPointerLevel() returns the same result as IsConstant(0) (i.e the final type will be checked).

Returns
true if the member type is constant, false otherwise.
