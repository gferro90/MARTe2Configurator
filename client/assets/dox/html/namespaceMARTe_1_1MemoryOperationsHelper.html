Detailed Description

Namespace which defines functions for managing blocks of memory.

The functions of this namespace allows the user to make operations like copying, moving, etc, on memory blocks.

Converts the memory from a flat to an interleaved representation for a specified number of samples.

Assume for instance an originSource with 2 samples organised as [(uint32)counter_1, counter_2][(uint16)x_1, x_2][(uint8)y_1, y_2]. Giving the following parameters:
packetMemberSize[] = {4, 2, 1}
packetByteSize = 7
numberOfPacketMembers = 3
numberOfSamples = 2
Then the output in originDest will be [(uint32) counter_1, (uint16) x_1, (uint8) y_1][(uint32) counter_2, (uint16) x_2, (uint8) y_2]. param[in] originSource the source memory area param[in] originDest the destination memory area param[in] beginIndex is the start index in the packetOutputChunkSize vector param[in] packetMemberSize the size of each member in the sample (i.e. packet) definition param[in] packetByteSize the size of the sample in bytes param[in] numberOfPacketMembers the number elements of each samples (i.e. packet, i.e. the number of elements in the packetMemberSize) param[in] numberOfSamples the number of samples

Precondition
originSource must be valid with a size greater than outputByteSize*numberOfSamples;
originDest must be valid with a size greater than outputByteSize*numberOfSamples;
Postcondition
originDest is filled with the re-organised data from flat to interleaved.

Converts the memory from an interleaved to flat representation for a specified number of samples.

Assume for instance an originSource with 2 samples organised as [(uint32) counter_1, (uint16) x_1, (uint8) y_1][(uint32) counter_2, (uint16) x_2, (uint8) y_2]. Giving the following parameters:
packetMemberSize[] = {4, 2, 1}
packetByteSize = 7
numberOfPacketMembers = 3
numberOfSamples = 2
Then the output in originDest will be [counter_1, counter_2][x_1, x_2][y_1, y_2]. param[in] originSource the source memory area param[in] originDest the destination memory area param[in] beginIndex is the start index in the packetMemberSize vector param[in] packetMemberSize the size of each member in the sample (i.e. packet) definition param[in] packetByteSize the size of each sample (i.e. of the packet) in bytes param[in] numberOfPacketMembers the number elements of each samples (i.e. packet, i.e. the number of elements in the packetMemberSize) param[in] numberOfSamples the number of samples

Precondition
originSource must be valid with a size greater than packetByteSize*numberOfSamples;
originDest must be valid with a size greater than packetByteSize*numberOfSamples;
Postcondition
originDest is filled with the re-organised data from interleaved to flat.
