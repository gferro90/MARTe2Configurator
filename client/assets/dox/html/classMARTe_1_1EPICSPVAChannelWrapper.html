◆ SetAliasAndField()

bool MARTe::EPICSPVAChannelWrapper::SetAliasAndField ( StructuredDataI &  data)

Registers the channel with a name given by data.Read("Alias") or data.GetName(), if the former does not exist.

Also tries to read the field name data.Read("Field"). If not set Field = value is assumed. The Field cannot be set as node as this would prevent the RealTimeApplicationConfigurationBuilder from properly expanding the structured types.

Parameters
[in]datathe structure to be registered against the channel.
