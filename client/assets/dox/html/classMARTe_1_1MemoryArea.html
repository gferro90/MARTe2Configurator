Detailed Description

Memory allocated area that behaves similarly to a StaticList with variable elements size.

The MemoryArea class facilitates the automatic management (allocation and freeing) of chunks of memory.

Initialises the memory area with a specific pre-allocated memory in input.

Parameters
[in]initialMemoryis the pointer a pre-allocated memory to be set
[in]initialSizeis the size of initialMemory
Returns
true if the memory area is empty, false if it is already initialised.
Precondition
GetMemoryStart() == NULL && GetMemorySize() == 0
Warning
The memory pointed by initialMemory is not copied but only wrapped. Functions like MemoryArea::Add(*) and MemoryArea::Free(*) can deallocate initialMemory. As a consequence make sure that initialMemory is not used outside the scope of MemoryArea.
