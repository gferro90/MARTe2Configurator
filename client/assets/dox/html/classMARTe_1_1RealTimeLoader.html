Detailed Description

Starts a generic MARTe RealTimeApplication from a given configuration stream.

The Loader parses (see Initialise) a given configuration stream and configures the RealTimeApplication. The Start method allows to trigger the Start of the application, either by specifying the first state or by sending a message to given destination (see Loader::Start).

StreamI &  configuration

Initialises the RealTimeApplication with the parameters specified in data and with the configuration stream (see Loader::Initialise).

If Loader::Initialise succeeds, a RealTimeApplication is search in the ObjectRegistryDatabase and, if found, the RealTimeApplication::ConfigureApplication is called.

Parameters
[in]datasee Loader::Initialise for other parameters:
  • FirstState (optional): the first state to be called in the RealTimeApplication when Start is called.
[in]configurationsee Loader::Initialise.
Returns
ErrorManagement::NoError if the Parser is specified, the configuration can be parsed, the ObjectRegistryDatabase can be Initialised with the parsed configuration and if the RealTimeApplication::ConfigureApplication is successful. An error is returned otherwise.

ErrorManagement::ErrorType MARTe::RealTimeLoader::Start ( )
virtual

Start the RealTimeApplication.

If FirstState was set, calls RealTimeApplication::StartNextStateExecution with this state. Otherwise Loader::Start is called.

Returns
ErrorManagement::NoError if the FirstState was set and RealTimeApplication::StartNextStateExecution or if FirstState was not set and Loader::Start succeeds. An error is returned otherwise.
