Detailed Description

Buffered stream (double buffer version).

This class offers a buffering mechanism for character streams.

◆ writeBuffer

BufferedStreamIOBuffer MARTe::DoubleBufferedStream::writeBuffer
protected

The write buffer.

It is used as a middle buffer between the stream and the output. For each write operation this buffer is filled completely and then the desired size is copied on the output.
