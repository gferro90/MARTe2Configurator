Detailed Description

Implementation of an interface to a NiSync timing device.

Follows an example of the configuration: +NiSyncDeviceInterface = { Class=NiSyncDevice DeviceType = PXI6683H DeviceNum = 0 +NiSyncSubModule = { Class = NiSyncClockGenerator Terminal = PFI1 ClockPeriod = 1000000000 DutyCycle = 500000000 StartTimeNanos = 1 Phase = 0 } } It can contain one of more NiSyncSubModule objects that performs operations on the timing board.
