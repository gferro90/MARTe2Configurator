Detailed Description

Memory mapped BrokerI implementation.

This class knows how to copy from/to a DataSourceI memory address to/from a GAM signal memory address. For each GAM signal, the signal name is searched in the provided DataSourceI (see Init) and the memory memory address of the signal retrieved using the GetSignalMemoryBuffer function. The information of each element to be copied is stored in a MemoryMapBrokerCopyTableEntry.

Initialises the MemoryMapBroker.

For each signal in the functionName, which wishes to use this MemoryMapBroker instance (i.e. IsSupportedBroker(class inhering from MemoryMapBroker) == true), the signal name is searched in the provided dataSourceIn (see Init) and the memory address of the signal retrieved using the GetSignalMemoryBuffer function. The information of each element to be copied is stored in a MemoryMapBrokerCopyTableEntry.

Parameters

[in]dataSourceInthe DataSourceI to be queried.
[in]functionNamethe name of GAM the to which this BrokerI is being allocated to.
[in]gamMemoryAddressthe base address of the GAM memory (where signal data is stored).
Returns
