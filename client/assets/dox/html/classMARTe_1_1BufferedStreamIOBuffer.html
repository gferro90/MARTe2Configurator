Detailed Description

BufferedStreamIOBuffer class.

This class inherits from IOBuffer specializing NoMoreSpaceToWrite and NoMoreDataToRead functions accordingly to the buffered stream requirements. In particular NoMoreSpaceToWrite acts as a flush on the stream and NoMoreDataToRead refills this buffer from the stream. The Resync function is also adjusts the stream position after a buffered read or write operation.

Remarks
Since this buffers has to read and write on the associated stream, the constructor needs a pointer to such stream.

If the stream is not NULL it calls stream->Flush, otherwise it calls IOBuffer::NoMoreSpaceToWrite. This implementation allows the buffer to access to the strean Flush function. In the stream Flush implementation the user must use the buffer NoMoreSpaceToWrite function instead of the buffer Flush to not cause a infinite call loop.

bool MARTe::BufferedStreamIOBuffer::Resync ( )
virtual

Adjusts the position of the stream.

This function is called from the stream after a read operation because the position was shifted forward (+bufferSize) due to the refill. Calls Seek moving the cursor back (-UsedAmountLeft).

Returns
false if the stream seek fails.
