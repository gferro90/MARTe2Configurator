Detailed Description

A basic implementation of a stream which allows reading and writing to an underlying unbounded string, which can grow dynamically.

◆ RelativeSeek()

bool MARTe::StreamString::RelativeSeek ( const int64  deltaPos)
virtual

Moves within the stream relative to current location.

Checks that the final position is >= 0 and <= UsedSize, then moves the cursor.
-If the final position is < 0 moves the cursor at the beginning.
-If the final position is > UsedSize moves the cursor at the end.

Parameters
[in]deltaPosis the gap from the current position.
Returns
false if the position falls out of bounds or if deltaPos is outside the int32 range of values (supported by IOBuffer::RealtiveSeek(*)).
Precondition
(deltaPos <= MAX_INT32) && (deltaPos >= MIN_INT32)
