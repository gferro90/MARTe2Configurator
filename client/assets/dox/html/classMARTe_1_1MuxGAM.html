Detailed Description

Implement a multiplexer using as many control signals (called selectors) as outputs.Each output is controlled by a single selector.

This GAM has two conceptual inputs: input signals which are the signals routed to the outputs and selectors which are the control signals. Notice the difference between inputs and input signals (inputs = input signals + selectors).
The GAM copies the input signals (indicated by the selectors) to the outputs. Several types of data and single/array elements are supported, however all the input and output signals must have the same type and the same number of elements. The selectors are always uint32 and the number of elements could be or 1 or the same than the input and output signals.

When the number of selector elements is 1,for a giver input signal array, all the elements will be routed to the same output. On the other hand, if the number of selector elements is equal to the number of input signal elements the routed path (input-output) can be changed for each element of the array.

Any number of inputs/outputs are allowed with the following restrictions:

  • The number of selectors signals must be equal to the number of outputs.
  • The number of output signals <= number of input signals.
  • The minimum number of outputs is 1.
  • The minimum number of input signals is 2. Hence the minimum number of inputs is 3 (1 selector and 2 input signals).
  • The maximum number of inputs is limited by the uint32 size (2^32-1 inputs).


The type supported by the GAM are:

  • SignedInteger8Bit
  • SignedInteger16Bit
  • UnsignedInteger16Bit
  • SignedInteger32Bit
  • UnsignedInteger32Bit
  • SignedInteger64Bit
  • Float32Bit
  • Float64Bit

As an example imagine a MuxGAM with 4 inputs (2 input signals, 2 selectors) and 2 output signals. Inputs and outputs are named I0, I1, S0, S1, O0 and O1. The S0 controls the O0 and the S1 controls the O1. With the previous conditions the following functional table could be done:

S0S1O0O1
0 0 I0I0
10I1I0
0 1I0I1
1 1 I1I1


As can be observed the same input can be connected to several outputs.
