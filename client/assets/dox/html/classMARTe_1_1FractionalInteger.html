Detailed Description

template<typename baseType, uint8 numberOfBits>
class MARTe::FractionalInteger< baseType, numberOfBits >

An integer wrapper class parameterisable by bit size.

This template class is an helper used to define integer types with non-standard bit sizes, such as uint3 or uint63.

Values which are greater than the maximum (or smaller than the minimum) allowed in a numberOfBits size will be saturated, thus avoiding any kind of rollover problem.

The cast operator is overridden so that the class will behave as a basic type for mathematical operations.

Template Parameters
baseTypethe standard type which is used as a base for the new type
numberOfBitsthe actual size of the new type
Remarks
numberOfBits must be less than the number of bits of baseType (e.g. less than 64 for a uint64)
