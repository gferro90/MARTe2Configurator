Detailed Description

A LoggerConsumerI which outputs the log messages to a console instance.

The configuration syntax is (names are only given as an example): +ConsoleLogger = { Class = ConsoleLogger Format = ItOoFm //Compulsory. As described in LoggerConsumerI::LoadPrintPreferences PrintKeys = 1 //Optional. As described in LoggerConsumerI::LoadPrintPreferences }
