Detailed Description

Registers a structure as defined by a StructuredDataI.

Each node is a structure member and shall have the field Type defined. The name of the Object is the name of the structure to register.

The field NumberOfElements allows to define multi-dimensional arrays of structures. The Type can be a BasicType or any structured Type (possibly also defined by another IntrospectionStructure). An example of a possible collection of structures would be (names only given as examples): 

  +MyTypes = {
    Class = ReferenceContainer
    +MyStructEx1 = { //name of the structured type to register.
      Class = IntrospectionStructure
      Field1 = {
        Type = uint32
        NumberOfElements = 1
      }
      Field2 = {
        Type = float32
        NumberOfElements = {3, 2} //3x2 matrix
      }
    }
    +MyStructEx2 = { //name of the structured type to register.
      Class = IntrospectionStructure
      Field1 = {
        Type = MyStructEx1
        NumberOfElements = {3, 2, 1} //3x1x2 matrix of MyStructEx1
      }
    }
  }

Registers a structure as defined by a StructuredDataI tree.

The StructuredDataI is recursively traversed and an equivalent structure constructed based on the leaf types. Each node will be registered as an individual structure. If the node contains a leaf named typeIdentifier, this will be used as the type of the structure to be associated to that node.

Example: 

  A1 = {
    c1 = 2
    A2 = {
      d2 = 2.0
    }
    A3[0] = {
      f1 = -1
    }
    A3[1] = {
      f1 = 2
    }
  }

Is registered as: 

  struct A2 {
    float32 d2;
  };
  struct A3 {
    int32 f1;
  };
  struct A1 {
    uint32 c2;
    A2 A2;
    A3 A3[2];
  };

If typeIdentifier had been set to e.g. RegisterType:

  A1 = {
    RegisterType = MainStruct
    c1 = 2
    A2 = {
      RegisterType = SubStruct
      d2 = 2.0
    }
    A3[0] = {
      RegisterType = SubStruct2
      f1 = -1
    }
    A3[1] = {
      f1 = 2
    }
  }

Would be registered as: 

  struct SubStruct {
    float32 d2;
  };
  struct MainStruct {
    uint32 c2;
    SubStruct A2;
    SubStruct2 A3[2];
  };
Parameters
[in]inthe structure to be registered, already moved inside the parent node to be registered (i.e. in.GetName() should return the type of the parent structure to register).
[in]typeIdentifiera keyword which allows to detect type definitions inside the nodes.
Returns
true if the structure can be successfully registered.
