Detailed Description

A group of GAMs sharing the same context.

This class allows GAMs to share a common context. GAMs which are referenced by a GAMGroup will have their SetContext method called.

The syntax in the configuration stream has to be:

+GAMGroup_name = {
Class = GAMGroup
GAM_name = {
Class = GAM
...
}
...
}

and it has to be contained in the [RealTimeApplication].+Functions.[?ReferenceContainer?] declaration.

bool MARTe::GAMGroup::Initialise ( StructuredDataI &  data)
virtual

Explores the StructuredDataI in input and builds Objects storing their References.

The Object will be built only if the node name in the data tree has the special symbol '+' or '$' at the beginning. The symbol '$' marks the node as a domain and will be used for relative researches by path in ObjectRegistryDatabase::Find(*)
