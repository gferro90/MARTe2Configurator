Detailed Description

Namespace which defines the available operating modes for a console.

The modes are defined as constants of type FlagsType. They are meant to be used for configuring the operating mode of consoles.

◆ Default()

const FlagsType MARTe::BasicConsoleMode::Default ( 0u  )

Default operation mode.

In this mode users have to explicitly feed a new line to write to the console output, CTRL-C will raise an exception and kill the underlying program and paging will not be enabled.
