Detailed Description

Starts a generic MARTe application from a given configuration stream.

The Loader parses (see Initialise) a given configuration stream and sends (see Start) a Message to a provided destination.

ErrorManagement::ErrorType MARTe::Loader::Configure ( StructuredDataI &  data,
StreamI &  configuration 
)
virtual

Initialises the Loader with the parameters specified in data and with the configuration stream.

Parameters
[in]datathe loader parameters:
  • DefaultCPUs (optional): sets the threads defaults CPUs (see ProcessorType::SetDefaultCPUs);
  • Parser: the type of parser to be parse the configuration as one of:cdb, xml and json;
  • MessageDestination (optional): the name of the Object that will receive the message when Start is called;
  • MessageFunction (optional, but compulsory if MessageDestination is set): the name of the Function to be called in the MessageDestination.
[in]configurationthe MARTe configuration stream to be loaded (and parsed using the Parser defined above).
Returns
ErrorManagement::NoError if the Parser is specified, the configuration can be parsed and if the ObjectRegistryDatabase can be Initialised with the parsed configuration. An error is returned otherwise.
