Detailed Description

Implementation of the chunked body send for the HTTP protocol.

See also
HttpService.

The chunked transfer encoding requires that the body is divided in chunks with a size that must be sent before the chunk itself. For instance, if the HTTP body to be sent is HelloWorld and the maximum chunk size is 4, the following must be sent:
