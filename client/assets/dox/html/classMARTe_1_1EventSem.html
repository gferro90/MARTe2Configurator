Detailed Description

Event semaphore.

The EventSem offer the possibility to block any number of threads in a barrier. This barrier is lowered by calling the Reset method and raised by the Post method. Threads are blocked in the barrier by calling one of the Wait methods. Once the barrier is raised all the threads are allowed to concurrently proceed.
