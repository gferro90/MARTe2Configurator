[in]modeToSetany ored combination of ReferenceContainerFilterMode::PATH, ReferenceContainerFilterMode::RECURSIVE, ReferenceContainerFilterMode::REVERSE and ReferenceContainerFilterMode::REMOVE.

bool MARTe::ReferenceContainerFilterObjectName::Test ( ReferenceContainer &  previouslyFound,
Reference const &  referenceToTest 
)
virtual

Tests if the name of the object hold by referenceToTest is compatible with the address.

This method will return true when the name of object hold by referenceToTest is the same of the last node in the address specified in the constructor. If a full path was specified as the address, this method will return true if, and only if, all the elements in previouslyFound where found in the exact same order set in the constructor address. Each time this method finds a matching reference the number of occurrences will be incremented.

Parameters
[in]previouslyFoundthe history of the path found so far.
[in]referenceToTestthe reference to test (where the name of object will be extracted from).
Returns
true if the name of the object hold by referenceToTest is compatible with the address.
